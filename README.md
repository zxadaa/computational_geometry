# Computational Geometry

The repository contains implementations of algorithms covered in the Computational Geometry university course. Laboratory resources and the base visualizer are provided by the [BIT Scientific Group](https://github.com/aghbit/Algorytmy-Geometryczne).


## Repository Structure

* **`labs/`** - Inside each `labX/` folder, you will find the `main.ipynb` Jupyter Notebook with the algorithm logic and a `report.pdf` documenting the results.
* **`tests/`** - Datasets and edge-case scenarios used to verify the correctness of the implementations.
* **`visualizer/`** - Tools used for rendering points, segments, polygons, and step-by-step algorithm animations.
